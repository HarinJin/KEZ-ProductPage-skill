# design-dna / 이미지 배치 — 판독·모방·자르기 vs 채우기 (image placement)

> 이 문서는 사진을 화면에 앉히기 **직전**의 두 판단만 다룬다: ① 주어진 자료·레퍼런스 이미지를 **판독해 무엇을 모방할지** 뽑고, ② 그 이미지를 **자르기(대상 강조)로 전경에 둘지, 채우기(배경 무드)로 후경에 깔지**를 **의도로** 결정한다.
> 결정한 뒤의 처리(블렌드·그림자·텍스트와의 거리·크기·삽입 방식)는 → `06-image-treatment`. 전체 **페이지**의 디자인 언어를 실측 해독하는 것(레퍼런스 상세페이지 분석)은 → `00-interview` §1.5. 이 문서는 **이미지 한 장 단위**의 판독·배치 의도만 정한다.

---

## 1. 자료 이미지 판독 — "무엇을 모방하나"를 먼저 언어화

레퍼런스/자료 이미지가 주어지면 "비슷하게 만들자"로 넘어가지 말고, **베낄 요소를 항목으로 분해**해 관찰한다. 관찰이 언어화돼야 제작에 옮길 수 있다.

| 판독 항목 | 무엇을 보나 | 제작에 어떻게 쓰나 |
|---|---|---|
| 구도 | 대상의 위치(중앙/좌우/삼분할), 여백의 분포 | 프레임 안 대상 배치·여백 예약 |
| 광원·톤 | 빛의 방향, 전체 톤(따뜻/차가운·밝기) | 섹션 배경톤·다른 컷과의 톤 통일 |
| 배경 처리 | 아웃포커스(얕은 심도)·색면·텍스처 중 무엇 | 대상을 띄우는 방식, 뒤 레이어 무드 |
| 심도차(대상↔배경) | 대상과 배경이 얼마나 분리돼 있나 | 전경/후경 층 분리 설계 |
| 색 팔레트 | 지배색·강조색 | 브랜드 색과 충돌 여부 점검 |

**원리:** 모방은 "느낌"이 아니라 **관찰한 항목의 재현**이다. 눈대중으로 "비슷하게"는 매번 다른 결과를 낳는다 → 판독 결과를 근거로 배치·색·톤을 정한다. (전체 페이지 단위 해독은 → `00-interview` §1.5.)

---

## 2. 자르기(CROP) vs 채우기(FILL) — 의도로 가른다

이미지를 넣는 두 큰 갈래는 **"대상만 강조하려는가, 배경 무드를 깔려는가"**라는 의도로 갈린다. 이 의도가 삽입 방식과 레이어를 동시에 정한다.

| 의도 | 방식 | 레이어 | 왜 |
|---|---|---|---|
| **이미지 속 대상(상품)만 떼어 강조** | **자르기(CROP)** / 누끼 | **전경** | 대상을 격리해 앞에 세워야 시선이 대상에 꽂힌다. 주변은 잘라내거나 투명화 |
| **전체를 뒷배경 무드로** | **채우기(FILL / cover)** | **후경(배경 레이어)** | 배경은 섹션을 꽉 채워 분위기를 만드는 것이라 잘라 강조할 대상이 없다. 여백 없이 덮는다 |
| 카드 안 근거컷(반복 규격) | 프레임에 **채우기** + 라운딩 클립 | 인라인 | 여러 장을 같은 박스 규격으로 통일해 리듬(→`06` §2①·§4) |

**핵심 한 줄:** **대상 강조 = 자르기(전경) / 배경 무드 = 채우기(후경).**

**보조 원칙**
- **한 섹션에 둘 다** 필요하면 층을 나눈다: 배경=채우기(후경 무드) + 대상=자르기·누끼(전경 강조). 배경 위에 대상을 얹는 히어로의 기본 구조.
- **배경(채우기) 위에 텍스트**가 오면 가독성은 **크롭이 아니라 대비(스크림·그라데이션)**로 확보한다(→`06` §6). 배경을 더 잘라내서 글자를 살리려 하지 말 것.
- 배경에서 **특정 부위가 특정 위치에 반드시 보여야** 하면(예: 상단에 텍스트존 확보) 채우기 대신 **자르기로 크롭 위치를 고정**하거나, 애초에 그 여백을 포함해 이미지를 준비/생성한다.
- **생성형 이미지 준비 지침**: full-bleed로 쓸 이미지는 (a) **주인공 상하에 단색 여백존**(텍스트 자리)을 의도적으로 확보, (b) **타깃 섹션 비율로 세로 길게** 생성(크롭 손실↓), (c) **그림자 금지** — 누끼·주인공 이미지엔 drop-shadow 없이(→`06` §3) 프롬프트에 "no drop shadow" 명시. 부양은 회전·겹침·여백으로.

## 3. 배경 무드 이미지(채우기) 운용 세부

- **섹션 배경색 = 이미지 지배톤과 연속**되게 잡는다 — 이미지와 배경 색면이 이어져 "한 장"으로 읽힌다.
- 그리드·도트·종이 같은 **텍스처 배경은 프레임보다 크게**(음수 오프셋 오버블리드) 깔아 섹션 이음새를 없앤다(→`06` §6b).
- 배경은 **저대비**로 — 콘텐츠보다 튀면 그건 배경이 아니라 방해다.
- 전면 배경은 자르지 않는 **full-bleed 통짜**가 기본(→`06` §2③).

## 4. 실측으로 반복 확인된 배치 패턴 (레퍼런스 해독 결과 반영)

기존/타깃 상세페이지를 실측 해독하면(→`00-interview` §1.5) 아래 배치 패턴이 반복해 나타난다. §1~3 원칙의 구체적 적용례이자, 새 페이지에서 그대로 재사용하는 관용이다. **전부 브랜드 무관**(색·소재만 갈아끼움).

- **① 제품 히어로 = 중앙 + 소품 접지 + 색면 바닥.** 대표 제품컷(자르기·누끼)을 허공에 띄우지 말고 **화면 중앙(살짝 우 허용)**에 놓고, 주변에 소품 몇 개로 접지(grounding)한다. 바닥은 단색 색면(채우기)으로 깔아 제품이 그 위에 "놓인" 물성을 만든다. → 무대(색면)·주연(중앙 제품)·조연(소품)의 3층.
- **② 같은 모티프의 이중 역할 — 배경 채우기 + 전경 자르기.** 하나의 소재(원물·재료·상징물)를 한 섹션에서 **배경 무드(채우기·후경)와 강조 컷(자르기·누끼·전경) 둘 다로** 쓴다. 위쪽엔 소재를 full-bleed로 깔아 "무엇으로 만들었나"를 무언으로 말하고, 본문 옆엔 같은 소재를 잘라 띄워 근거를 짚는다. → **자르기/채우기는 이미지별 택1이 아니라 모티프별로 층을 나눠 병용**하는 것(§2 "한 섹션에 둘 다"의 실증).
- **③ content-as-texture 배경.** 실제 자료(후기 캡처·기사·인증서 등)를 **흐리게 저대비 full-bleed(채우기)**로 깔아 "증거의 벽" 무드를 만들고 그 위에 또렷한 본문/카드를 얹는다. 배경은 판독이 목적이 아니라 물량감·신뢰의 분위기가 목적.
- **④ 히어로 북엔드.** 긴 섹션(리뷰 등)의 처음과 끝을 **같은 대표 제품컷**으로 감싸 시각적 닫음(closure)을 준다.
- **⑤ 배경 톤 연속.** 배경 이미지(채우기)의 지배톤과 섹션 배경 색면을 **같은 계열로 이어** 이미지와 색면이 한 장으로 읽히게 한다(→§3).

---

## 5. 체크리스트

1. **이 이미지의 의도?** 대상 강조 → **자르기(전경)** / 배경 무드 → **채우기(후경)**.
2. **판독한 모방요소**(구도·광원·톤·심도·팔레트)를 항목으로 반영했나(§1).
3. **한 섹션에 대상+배경**이면 전경(자르기)·후경(채우기)로 층 분리했나.
4. **배경 위 텍스트**면 스크림/그라데이션으로 대비 확보(→`06` §6). 크롭으로 때우지 않기.
5. 결정 뒤 **처리 기법**(블렌드·그림자·거리·크기·삽입 방식)은 → `06-image-treatment`.
6. **히어로 제품**은 중앙+소품 접지+색면 바닥인가, **같은 모티프를 배경(채우기)+전경(자르기) 병용**했나 (§4).

바뀌는 것은 이미지 내용뿐, **의도(대상 강조/배경 무드) → 자르기/채우기 매핑은 고정**이다.
